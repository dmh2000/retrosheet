### Gamelogs

- cd to csv-transform/csv-transform-games
- npm install
- node index.js [directory with retrosheet gamelog .TXT files] [output directory]
- "$ node index.js ~/projects/baseball/data/gamelogs/csv ~/projects/baseball/data/gamelogs/json"
